  ## centOS
  - 필수적인 기본 명령어
      - ls : 파일 리스트 출력
      - cd ~ : 자신(운영체제 사용자)의 개인폴더로 이동
        - cd : 뒤로가기
      - rmdir ./폴더명 : 디렉토리 삭제
      - mkdir ./폴더명 : 디렉토리 생성
      - vim 문서파일명 : 문서파일을 수정하거나 생성
      - rm 파일명 : 파일을 지운다.
